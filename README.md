## Pre-Requisite

**Python:** >= 3.8

## Getting Started

Install using `make`

```bash
make install
```

If for some reason it does not work,
here are the manual steps to install

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install requirements.txt
```
